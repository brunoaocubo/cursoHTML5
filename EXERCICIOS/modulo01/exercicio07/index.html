Conhecendo Tags de Formatação

Palavras em Negrito/Destaque

HTML4 = <b>
HTML5 = <strong>

Palavras em Itálico/Ênfase

HTML4 = <i>
HTML5 = <em>

Texto grande e pequeno

No HTML4, a tag <big> era utilizada para deixar o Texto Grande, no HTML5 tornou-se obsoleto e será necessário utilizar o CSS caso deseje aumentar. Já com Textos Pequenos, podemos utilizar a tag <small>.

Texto Deletado

HTML4 = <strike>
HTML5 = <del>

Texto Inserido

HTML4 = <u>
HTML5 = <ins>

Texto Marcado

Podemos marcar um texto utilizando a tag <mark>.
Exemplo: Texto Marcado

Texto Sobrescrito

Para um texto sobrescrito como: (4 4 = 256) utiliza-se a tag <sup>.

Texto Subescrito

Para um texto Subescrito como: H2O utilizamos a tag <sub>.
