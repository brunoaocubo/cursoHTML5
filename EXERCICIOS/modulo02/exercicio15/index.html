Cores em CSS

Exercicio de cores em css, hora de praticar.

Este título está colorido usando cores pelo nome. Ex:(blue,red)

Este está usando cores pelo código Hexadecimal. Ex:(#FFFF)

Título utilizando cores pelo código RGB. Ex:RGB(red, green,blue)

Titulo utilizando cores pelo código HSL. Ex:HSL(hue, saturation, light)

As cores resultantes são exatamente as mesmas, mas o formato de representação muda conforme o padrão escolhido.

Também é possível usar transparência, usando o formato RGBA (red, green, blue, alpha).

Neste título, o fundo azul está com a transparência em 50%
